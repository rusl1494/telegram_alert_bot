## Description
Simple Telegram alert bot built with Python.

The bot sends automated notifications to a Telegram chat on a schedule.  
Designed as a lightweight example of Python automation and messaging integration.

## Features
- Telegram message sending
- Scheduled alerts
- Environment-based configuration

## Tech Stack
Python, Telegram Bot API, Requests

## Usage
1. Create a Telegram bot and obtain a bot token
2. Set environment variables
3. Run the script

## Notes
This project is provided for demonstration and portfolio purposes.
